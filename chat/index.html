Ask Open Source Europe

A public assistant that searches the official OSE documentation and answers with sources you can verify. Fiscal hosting, onboarding, policies, processes — ask in your own words.

Use it in your own tools — or run it locally

The OSE knowledge base speaks MCP. Wire it into your editor, install the Claude Code plugin, or clone the repo and run this whole chat on your machine.

Any MCP client — .mcp.json 
{
  "mcpServers": {
    "ose-knowledge": {
      "type": "stdio",
      "command": "uvx",
      "args": ["ose-knowledge-mcp"]
    }
  }
}

One package, zero install — the whole knowledge base runs on your machine.

Claude Code plugin 
/plugin marketplace add opensourceeurope/ose-knowledge-mcp
/plugin install ose-knowledge@ai
Run this chat locally — fully offline via Ollama 
git clone https://github.com/opensourceeurope/ose-knowledge-mcp.git
cd ose-knowledge-mcp
./scripts/run-local.sh

Needs uvx, npm, python3 and a running Ollama. Starts the MCP server, the agent function and this chat UI on localhost — no API keys, nothing leaves your machine.
